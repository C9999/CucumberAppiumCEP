## Appium + Cucumber

Automação criada usando Appium, e Cucumber para a escrita de cenários BDD.

## Ferramentas
- Appium server v1.6.4
- Script de teste feito em Java;
- Cucumber
- Selenium;
- JUnit;
- Android AVD, versão 7.1.1;
- Maven para gerenciamento de dependências do projeto
